# cRPC
Remote Procedure Call Demo Written in Pure C

# Getting Started
1. Build the binaries
    ```bash
        $ make
    ```
2. Run the server
    ```bash
        $ ./server
    ```
3. Start a new Terminal session and run the client
    ```bash
        $ ./client
    ```
